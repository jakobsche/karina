Collaboration at a github project

- Create a personal account
- verify your email address
- Go to https://github.com/jakobsche/karina
- click "Fork" near the right top corner of the page. This copies the project to your account
- git clone https://github.com/nancyperlt/karina.git
- edit the project in the new local folder karina
- execute "git status"
- changed files are listed in red
- add the changes to your local repository with "git add ..." (The following parameters are the names of the changed files)
- git config --global user.email "nancyperlt3@gmail.com"
- git config --global user.name "Nancy Perlt"
- git commit -m "French translation updated"
- git status
